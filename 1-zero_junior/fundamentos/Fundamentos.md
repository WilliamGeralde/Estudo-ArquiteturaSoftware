# Fundamentos

Nesse tópico será estudado os fundamentos iniciais da computação. Desde o funcionamento básico de um computador até o funcionamento dos sistemas operacionais. O objetivo é entender como um computador funciona por dentro e como os programas rodam na memória, para assim ter uma base sólida para o estudo de linguagens de programação, desenvolvimento de software e principalmente arquitetura de software.
